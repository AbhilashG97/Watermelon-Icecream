# Annotation Basics

## Format of a Java Annotation

A Java annotation can be of the following types - 

1. ```java 
    @Entity
    ``` 
1. ```java 
    @Entity(element = value)
    ``` 
    or
     
    ```java
    @Entity(value)
    ```
1. ```java
    @Entity(
        element1 = value1, 
        element2 = value2
    )
   ```
